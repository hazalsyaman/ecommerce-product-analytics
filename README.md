# E-commerce Product Analytics — Beginner SQL Portfolio Project

## Goal
Analyze an e-commerce order dataset with SQL and turn the findings into product and business recommendations.

This project is designed to demonstrate:
- SQL fundamentals
- KPI and product analytics thinking
- Customer and channel analysis
- Clear communication of findings

## Business Questions
1. What is the total completed revenue?
2. How does revenue change by month?
3. Which categories and products generate the most revenue?
4. Which acquisition channels bring the most revenue?
5. Which cities perform best?
6. What are the cancellation and return rates?
7. How many customers placed more than one completed order?
8. Do discounts appear to improve completed revenue or create operational risk?
